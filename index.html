Karte 1 – Teil (a)

Wir betrachten das Zentralpotential \[ V(r) = -\frac{c}{r^\alpha}, \qquad r = |\vec r|. \] Zeige, dass das zugehörige Kraftfeld \(\vec F(\vec r)\) den Ausdruck \[ \vec F(\vec r) = -\frac{\alpha\,c\,\vec r}{r^{\alpha+2}} \] besitzt.

Klick auf die Karte für die Antwort

Antwort zu Teil (a)

Für ein konservatives Feld gilt \[ \vec F(\vec r) = -\nabla V(r). \] Da \(V\) nur von \(r = |\vec r|\) abhängt, können wir schreiben \[ \nabla V(r) = \frac{\mathrm d V}{\mathrm d r}\, \frac{\vec r}{r}. \] Mit \[ V(r) = -c\,r^{-\alpha} \] erhalten wir \[ \frac{\mathrm d V}{\mathrm d r} = \alpha\,c\,r^{-(\alpha+1)}. \] Damit folgt \[ \nabla V(r) = \alpha\,c\,r^{-(\alpha+1)} \frac{\vec r}{r} = \alpha\,c\,\frac{\vec r}{r^{\alpha+2}}. \] Also \[ \vec F(\vec r) = -\nabla V(r) = -\frac{\alpha\,c\,\vec r}{r^{\alpha+2}}. \] Damit ist gezeigt: \[ \boxed{ \vec F(\vec r) = -\dfrac{\alpha\,c\,\vec r}{r^{\alpha+2}} }. \]

Antwort – klick wieder auf die Karte, um zurückzugehen

Karte 2 – Teil (b)

Für das Kraftfeld \[ \vec F(\vec r) = -\frac{\alpha\,c\,\vec r}{r^{\alpha+2}} \] diskutiere, ob folgende Größen erhalten sind:
(i) Energie \(E\)
(ii) Impuls \(\vec p\)
(iii) Drehimpuls \(\vec L = \vec r \times \vec p\).

Hängt das vom Wert von \(\alpha\) ab?

Klick auf die Karte für die Antwort

Antwort zu Teil (b)

(i) Energie \(E\)
Die Lagrange-Funktion \[ L = T - V = \frac12 m\dot{\vec r}^{\,2} - V(r) \] hängt nicht explizit von der Zeit ab. Daher ist die Gesamtenergie \[ E = T + V = \frac12 m\dot{\vec r}^{\,2} + V(r) \] erhalten: \[ \boxed{E \text{ ist erhalten.}} \]

(ii) Impuls \(\vec p\)
\[ \vec p = m\dot{\vec r}, \qquad \dot{\vec p} = m\ddot{\vec r} = \vec F(\vec r). \] Da \(\vec F(\vec r) \neq 0\), ändert sich \(\vec p\) mit der Zeit. Nur wenn \(\vec F = 0\) wäre, wäre \(\vec p\) erhalten. Daher: \[ \boxed{\vec p \text{ ist im Allgemeinen nicht erhalten.}} \]

(iii) Drehimpuls \(\vec L\)
Das Potential ist zentral, d. h. \(V = V(r)\). Die Kraft zeigt daher immer radial: \[ \vec F(\vec r) \parallel \vec r. \] Das Drehmoment \[ \vec \tau = \vec r \times \vec F(\vec r) = \vec 0. \] Aus \(\dot{\vec L} = \vec\tau\) folgt \[ \dot{\vec L} = \vec 0 \quad\Rightarrow\quad \vec L = \text{konstant}. \] Also: \[ \boxed{\vec L \text{ ist erhalten.}} \]

Abhängigkeit von \(\alpha\)
Alle obigen Argumente benutzen nur, dass das Feld konservativ und zentral ist. Der konkrete Exponent \(\alpha\) spielt keine Rolle: \[ \boxed{ \text{Die Erhaltungseigenschaften sind unabhängig von }\alpha. } \]

Antwort – klick wieder auf die Karte, um zurückzugehen

Karte 3 – Teil (c)

Zeige, dass die Bewegung in diesem Zentralfeld immer in einer Ebene stattfindet, deren Normalenvektor parallel zum Drehimpuls \(\vec L\) verläuft.

Klick auf die Karte für die Antwort

Antwort zu Teil (c)

Der Drehimpuls ist \[ \vec L = \vec r \times \vec p \] und im Zentralpotential erhalten: \[ \dot{\vec L} = 0 \Rightarrow \vec L = \text{konstant}. \] Betrag und Richtung von \(\vec L\) bleiben also unverändert.

Zu einem Zeitpunkt \(t_0\) spannen \(\vec r(t_0)\) und \(\vec p(t_0)\) eine Ebene auf. Der Drehimpuls \(\vec L\) steht definitionsgemäß senkrecht auf dieser Ebene.

Die Beschleunigung ist \[ \vec a = \ddot{\vec r} = \frac{1}{m}\vec F(\vec r) = -\frac{\alpha c}{m}\,\frac{\vec r}{r^{\alpha+2}}, \] also radial und damit ebenfalls in dieser Ebene.

Da \(\vec L\) konstant bleibt, bleibt auch die Ebene, die durch \(\vec r\) und \(\vec p\) aufgespannt wird, dieselbe. Die Bahn verlässt diese Ebene nie.

Damit: \[ \boxed{ \text{Die Bahn liegt vollständig in einer Ebene, deren Normalenvektor parallel zu }\vec L\text{ ist.} } \]
